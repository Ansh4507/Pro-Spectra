# 📊 ProSpectra – Your Placement Companion

ProSpectra is a vibrant, responsive MERN stack application designed to help students track, organize, and prepare for placement opportunities in an intelligent and structured way.

---

## 🚀 Features

- 🔐 **User Authentication** (Planned): Secure login/signup.
- 📝 **Add Company Info**: Name, eligibility, resume status, rounds, interview experiences, JD, revision topics.
- 🤖 **Chatbot Assistant**: Ask questions about added companies (LLM integration coming soon!).
- 💡 **Smart Filtering/Search**: Quickly find entries by keyword or status.
- 📈 **Dashboard & Analytics** (Upcoming): Visual insights into your placement journey.
- 📅 **Calendar Integration** (Planned): Google Calendar for interview scheduling.
- 📤 **File Uploads**: Resumes/JDs via Cloudinary (in progress).
- 📳 **Notifications**: Reminders for company visits and revisions.
- 🧠 **RAG-Enhanced Chatbot** (Planned): Intelligent conversation with embedded company data.
- 🎨 **Beautiful Responsive UI**: Glassmorphism + animations using Chakra UI + Framer Motion.

---

## 🛠 Tech Stack

### Frontend
- **React.js + Vite**
- **Chakra UI** – Styled components
- **Framer Motion** – Animations
- **Axios** – API calls
- **React Router** – Routing
- **Cloudinary** – (for media upload)

### Backend
- **Node.js + Express.js**
- **MongoDB Atlas** – Cloud database
- **Mongoose** – ODM
- **CORS, dotenv, body-parser**

## 🧪 How to Run Locally

### 📦 Prerequisites
- Node.js & npm
- MongoDB Atlas account
- Cloudinary account

### 🖥 Backend Setup

```bash
cd backend
npm install
# Add your MongoDB URI in .env
npm start
```

### 🖥 Backend Setup

```bash

cd frontend
npm install
npm run dev
